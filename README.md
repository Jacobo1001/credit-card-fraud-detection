# Credit Card Fraud Detection

This academic project focuses on analyzing financial transactions to detect fraudulent operations using machine learning techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Project Description
The dataset contains anonymized credit card transactions. The objective is to identify fraudulent transactions through data preprocessing, exploratory analysis, and classification models.

## Models Implemented
- Perceptron
- AdaBoost

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix

## Objective
To evaluate the performance of classification models in detecting fraudulent transactions and understand the challenges of imbalanced datasets.
